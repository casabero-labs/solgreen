# Calidad de datos

## Dimensiones

- completitud;
- unicidad;
- validez;
- consistencia;
- puntualidad;
- sincronización;
- trazabilidad.

## Reglas

- hueco mayor a 1,5 veces el intervalo esperado;
- duplicado exacto o conflictivo;
- timestamp fuera de orden;
- valor fuera de rango físico;
- señal congelada;
- contador acumulado decreciente;
- SOC incompatible con energía;
- signo contradictorio;
- balance imposible.

## Visualización

Los datos ausentes no se unen con una línea continua. Los datos interpolados usan trazo y etiqueta distintos.
