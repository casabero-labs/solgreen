# Roles y permisos

| Acción | Propietario | Técnico | Administrador | Analista |
|---|---:|---:|---:|---:|
| Cargar archivos | Sí | Opcional | Sí | No |
| Ver datos privados | Sí | Por invitación | Sí | Anonimizados |
| Ejecutar análisis | Sí | Sí | Sí | Sí |
| Solicitar interpretación IA | Sí | Sí | Sí | Sí |
| Confirmar causa | Sí, con técnico | Sí | No | No |
| Cambiar perfil de planta | Sí | Proponer | Sí | No |
| Gestionar proveedores IA | No | No | Sí | No |
| Generar informe compartible | Sí | Sí | Sí | Anonimizado |
| Eliminar original | Con política | No | Con auditoría | No |

Toda acción sensible genera registro de auditoría.
