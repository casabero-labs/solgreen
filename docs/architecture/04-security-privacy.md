# Arquitectura de seguridad y privacidad

- Auth y RBAC con Butterbase.
- RLS por `plant_id`.
- storage privado.
- claves en Infisical.
- seriales cifrados/redaccionados.
- auditoría append-only.
- proveedores IA detrás de gateway.
- política de minimización y retención.
- exportaciones con marca de clasificación.
- backups cifrados y restauración probada.
