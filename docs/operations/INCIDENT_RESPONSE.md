# Respuesta a incidentes de la planta

## Niveles

- Observación.
- Seguimiento.
- Revisión técnica prioritaria.
- Acción inmediata.

## Flujo

1. Preservar datos y capturas.
2. No desactivar protecciones.
3. Confirmar continuidad de servicio y riesgos visibles.
4. Descargar log nativo del inversor/BMS.
5. Crear episodio en Solgreen.
6. Medir variables físicas requeridas.
7. Registrar diagnóstico y acción.
8. Verificar no recurrencia.
9. Cerrar con evidencia.

Solgreen no sustituye procedimientos de seguridad eléctrica.
