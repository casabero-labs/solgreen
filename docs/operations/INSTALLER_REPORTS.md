# Reportes para instaladores

Todo reporte debe incluir:

- identificación redaccionada;
- periodo y fuentes;
- calidad de datos;
- cronología;
- tabla antes/durante/después;
- gráficas con unidades;
- hechos y cálculos;
- hipótesis separadas;
- códigos solo si fueron observados o documentados;
- pruebas solicitadas;
- mediciones requeridas;
- riesgos;
- anexos y hashes;
- firma/estado de revisión.
